# Vecka 2 – CSS & Design

## Syfte
Vecka 2 handlar om att bygga upp webbplatsens layout och visuella struktur med CSS.  
Fokus ligger på Grid och Flexbox samt att skapa ett enhetligt utseende för alla sidor.

enkel JavaScript-logik byggs denna vecka.

---

## Hur ska vi tänka denna vecka?
- Struktur före detaljer
- Grid för hela sidan
- Flexbox för innehåll inuti sektioner
- Samma header och footer på alla sidor
- Enkel och konsekvent design

---

## Vad gör vi under veckan?
- Bygger sidans grundlayout med CSS Grid
- Skapar header med Flexbox
- Lägger till navigation (meny)
- Gör headern sticky
- Skapar footer med tre sektioner
- Sätter färger och grundläggande typografi

---

## Pseudokod – hur sidan är uppbyggd

SIDLAYOUT
  BODY använder GRID
    HEADER
    MAIN
    FOOTER

HEADER
  använder FLEX
    LOGO
      när klick → gå till startsidan
    MENU
      länk: Hem
      länk: Projekt
      länk: Kontakt
    DARK MODE
      endast visuell knapp (ingen logik)

FOOTER
  använder GRID med 3 kolumner
    KOLUMN 1
      plats / adress
    KOLUMN 2
      sociala länkar
    KOLUMN 3
      kontakt (mail, telefon)
  COPYRIGHT placeras längst ner

---

## Vad ska vara klart efter vecka 2?
- Grundlayout med CSS Grid (header, main, footer)
- Header med Flex-meny + sticky-funktion
- Footer med 3 sektioner (plats, social, kontakt) + copyright längst ner
- Enhetligt färgtema och grundläggande typografi
- Samma header och footer på alla sidor

# Kursplan – Bootcamp för Fullstackutvecklare

## Översikt
Denna kurs ger en praktisk introduktion till webbutveckling med fokus på HTML, CSS, JavaScript och Git. Kursen avslutas med ett projekt där deltagaren bygger en personlig profil/CV-sida.

**Omfattning:** 30 YH-poäng  
**Längd:** 6 veckor  
**Studieform:** Praktisk, projektbaserad

---

## Kursmål
Efter avslutad kurs ska deltagaren kunna:
- Skapa strukturerade webbsidor med HTML
- Styla webbsidor med CSS
- Använda grundläggande JavaScript för interaktivitet
- Arbeta med Git och GitHub
- Bygga och publicera en enkel profil/CV-sida
- Arbeta enligt grundläggande agila principer

---

## Veckoplanering

### Vecka 1 – Introduktion & Webben
- Vad är webbutveckling?
- HTML-grunder
- Struktur och semantik
- Projektstart: planera CV-sidan

### Vecka 2 – CSS & Design
- CSS-grunder
- Layout (Flexbox/Grid)
- Responsiv design
- Design av CV-sidan

### Vecka 3 – JavaScript Grunder
- Variabler och funktioner
- DOM-manipulation
- Enkel interaktivitet
- Förbättra CV-sidan med JS

### Vecka 4 – Git & Samarbete
- Git-grunder
- GitHub
- Versionshantering i projekt
- Strukturera kod och mappar

### Vecka 5 – Projektarbete
- Vidareutveckling av CV-sidan
- Tillgänglighet & enkel UX
- Förberedelse för publicering

### Vecka 6 – Slutförande & Presentation
- Slutjusteringar
- Publicering
- Presentation av projekt
- Reflektion och feedback

---

## Projekt – Profil/CV-sida
Deltagaren ska bygga en personlig webbplats som innehåller:
- Presentation
- Kompetenser
- Erfarenhet/utbildning
- Kontaktinformation

---

## Bedömning
- Aktivt deltagande  
- Versionshantering med Git  
- Färdigt och fungerande projekt  
- Presentation av arbetet  

**För att bli godkänd krävs** att deltagaren har deltagit aktivt under kursen samt lämnat in ett fungerande profil/CV-projekt som versionshanterats med Git och presenterats vid kursens slut.
